# DataSage AI

## Overview

DataSage AI is an AI-powered business analytics dashboard built using Python, Streamlit, Pandas, and Matplotlib.

The application helps users upload CSV datasets, analyze sales data, generate business insights, identify risks, and create downloadable business reports.

## Features

* CSV File Upload
* Dataset Preview
* Dataset Summary
* Business Insights
* KPI Dashboard
* Bar Chart Visualization
* Pie Chart Visualization
* AI Insights
* Risk Alerts
* Products Below Average Analysis
* Data Health Score
* Missing Values Analysis
* Natural Language Query Support
* AI Business Report Generation
* Downloadable Reports
